## Usage

A _versioning_ configuration of the `terraform-aws-s3-bucket` Module could look like this:

```hcl
module "versioning-example" {
  source  = "operatehappy/s3-bucket/aws"
  version = "1.1.1"

  name   = "oh-demos-versioning-example"

  versioning = {
    enabled    = true
    mfa_delete = false
  }
}
```
